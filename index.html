DJAlert

A DJAlert jQuery plugin for creating notification that auto dismiss after 5 seconds.

Demo.


Make custom notification

call DJAlert Function and pass 6 parameter

  1. message "Notification message"
  2. type "message type"
  3. background color and gradient: "color name "red" or "linear-gradient(315deg, #6b0f1a 0%, #b91372 74%)"
  4. notification icon : "SVG code, PNG and ETC"
  5. if custom pass : true (default false)
  6. Delay : default 5s only seconds like (3)
Example:

<button class="type" onclick="DJAlert(`Notification message`,`type`,`linear-gradient(315deg, #6b0f1a 0%, #b91372 74%)`, `<svg width='24' height='24' viewBox='0 0 32 32' id='icon' xmlns='http://www.w3.org/2000/svg'> <path fill='white' d='M29.4162,14.5905,17.41,2.5838a1.9937,1.9937,0,0,0-2.8192,0L2.5838,14.5905a1.9934,1.9934,0,0,0,0,2.819L14.5905,29.4162a1.9937,1.9937,0,0,0,2.8192,0L29.4162,17.41A1.9934,1.9934,0,0,0,29.4162,14.5905ZM21,18H11V14H21Z'/>`,true,3)"> Custom</button>

Note : Required jQuery > 3.6
